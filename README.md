# E-Commerce Sales Dashboard (Power BI)

## Project Overview

This project analyzes e-commerce sales data using Power BI. The dashboard provides insights into revenue, sales trends, product performance, city-wise sales, and category contribution.

## Dashboard Features

* Total Revenue Analysis
* Total Orders Tracking
* Quantity Sold Analysis
* Revenue by Category
* Revenue by City
* Monthly Revenue Trend
* Top 5 Products by Revenue
* Interactive Filters and Slicers

## Tools Used

* Power BI
* DAX
* CSV Dataset

## Key Insights

* Identified top-performing products and categories.
* Compared revenue across multiple cities.
* Analyzed monthly sales trends.
* Built an interactive dashboard for business decision-making.
